## 📖 Overview

**RoadMapAI** is an independent academic research project that proposes an **AI-powered personalized learning roadmap recommendation system**. The objective is to help learners choose the right technologies, identify skill gaps, and receive customized learning paths using **Large Language Models (LLMs)**.

Unlike traditional static technology roadmaps, RoadMapAI generates personalized recommendations based on a learner's existing skills, career goals, and preferred technology domain.

> **Note:** This repository contains an IEEE-style research paper created for academic learning and portfolio purposes. It is **not an IEEE-published paper**.
